My Notes

These are the HTML tags which I have learnt.

<li>
This tag defines a list term.
<div>
This tag defines a division or a section in an HTML document.
<tr>
This tag defines a row of cells in a table.
<footer>
This tag represents a footer for its nearest sectioning content.It typically contains information about the author of the section or copyright data.
<main>
This tag represents the dominant content of the body of a document.
